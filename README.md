# S-AES + XTS Encryption Project

## Description
This project implements Simplified AES (S-AES) with XTS mode.

## Features
- S-AES encryption/decryption
- XTS mode implementation
- File encryption
- Brute force attack simulation

## How to Run

### 1. Encrypt
python xts.py

### 2. Attack
python attack.py
